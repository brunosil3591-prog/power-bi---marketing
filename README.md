# Objetivo do projeto: 
Esse projeto tem como objetivo analisar tipo de publico e em qual bairro a maior parte dele reside, para que a gestora consiga direcionar estrategias de marketing para o publico alvo.
# Contexto do negocio:
O dashboard pode ser utilizado para identificar se as vendas são mais comuns entre homens ou mulheres, assim o marketing pode criar estratégias voltadas para o publico que compra mais, além disso, identificando onde a maior parte dos compradores moram a dona pode considerar abrir uma filial no bairro com maior numero de clientes. 
# Fonte do dados:
Dados ficticios / simulados <br>
Formato dos arquivos: CSV / Excel
# Principais visuais:
Cartões: Faturamento total, media de pet por cliente, quantidade de vendads <br>
Pizza: Vendas por genero de cliente
Barras: Faturamento por bairro
Linhas: Faturamento por data
